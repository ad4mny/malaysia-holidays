# Examples

This folder contains example usage files for the malaysia-holiday package.

## Running Examples

```bash
# Run basic example (JavaScript)
npm run demo

# Run state filtering example
npm run demo:state
```

## Example Files

- **example.js** - Basic usage examples in JavaScript
- **example-state.js** - State filtering examples
- **example.ts** - TypeScript usage example

## Usage

Each example demonstrates different features:

### example.js

- Get holidays for a specific year
- Get current year holidays
- Get next upcoming holiday
- Check if a date is a holiday
- Holiday distribution analysis

### example-state.js

- Filter holidays by state
- Compare holiday counts across states
- Find state-specific holidays
- Regional holiday analysis

### example.ts

- TypeScript type-safe usage
- Interface demonstrations
- Type definitions in action
